# Enterprise Network Architecture Framework (ENAF)

## Overview

The Enterprise Network Architecture Framework (ENAF) is a practical, vendor-neutral methodology for designing, transforming, governing, and operating modern enterprise network architectures.

ENAF bridges the gap between business strategy and technology implementation by providing a structured architecture lifecycle, reusable assets, and real-world guidance.
